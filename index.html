QCD Langage Web

Les informations contenu dans l’élément <head> sont affichées sur la page web ?
Les informations que contient l'en-tête ne sont pas affichées sur la page, ce sont simplement des informations générales à destination de l'ordinateur.
La balise <title> sert à créer des titres qui seront affichés dans la page web?
La balise <title> affiche le titre de la page dans la barre de titre du navigateur à destination de l'ordinateur
En xHTML strict, l'élément body peut contenir du texte ?
Le texte doit être encapsulé, par ex. dans un élément p de paragraphe.
Le footer doit apparaitre en haut de la page ?
Le footer doit apparaître en bas de la page.
Le liens avec le fichier css est contenu dans un balise <link> dans le header de la page ?
De plus il doit comporter la propriété rel="stylesheet".
